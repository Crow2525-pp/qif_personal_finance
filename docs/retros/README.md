# Session Retrospectives

Write a short retrospective here after non-trivial Claude sessions.

## Filename

Use `YYYY-MM-DD-topic.md`.

## Template

```markdown
# YYYY-MM-DD - Topic

## Goal

## Changed Files

## Verification

## Follow-ups
```

Keep retros operational and concise. The next session should be able to read the latest relevant retro instead of re-scanning the whole repository.
