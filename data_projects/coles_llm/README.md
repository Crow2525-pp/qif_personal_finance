# Coles LLM Data Project

This is the documented data-project slot for the Coles LLM shopper work.

The active local checkout currently exists as a separate nested Git repository at `coles-llm-shopper/`. It is intentionally not imported into this repository by the platform/data-project restructure, because doing so would mix another repository's source history and local data into this PR.

When this project is ready to be managed by the shared workspace, move or vendor only the intended source files into this folder and keep generated data, screenshots, caches, and credentials out of Git.
