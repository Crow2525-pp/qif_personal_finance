# Claude Memory Index

Use this directory for durable, typed project memory that should survive across sessions without bloating `CLAUDE.md`.

## Naming

- `project_*.md`: stable architecture, workflow, or domain facts.
- `feedback_*.md`: user preferences or corrections from real work.
- `reference_*.md`: compact summaries of large docs, dashboards, or investigations.
- `user_*.md`: user-specific workflow preferences that are safe to store.
